### 매개변수

-객체가 특정 행위를 수행하려면 부가적인 데이터 필요
타입과 이름 이 매개변수라고 함

```
- (id)eat;
```

여러번 부르면 이렇게 됨
```
- (id)eat:(id)food;
```



구현부 안에서만 쓰임

###구현

```
- (id)eat: (id)food {
NSLog(@"%@을 먹습니다", food);
return nil;
}
```


###호출
```
Person *me = [[Person alloc] init];
[me eat:@"사과"];
```


###메서드 정의
```
- (반환타입)메서드이름:(타입)내부이름
           외부이름:(타입)내부이름...;
```


###메서드 구현
```
- (id)runTo:(id)location bySpeed:(id)speed {
NSLog(@"%@로 달립니다. 속도는%@입니다", location, speed);
return nil;
}
```

###메서드 호출
```
Person *me = [[Person alloc] init];
[me runTo:@"부산" bySpeed:@100];
```
